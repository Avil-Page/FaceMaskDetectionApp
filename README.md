
# Real time face mask detection in Android 

![demo 1](demo/demo1.gif) ![demo 2](demo/demo2.gif) 


### Overview

Detech if person is wearing a face mask or not in real time using Android CameraX API and TensorFlow Lite.

### Bluetooth

Connect to HC-05 bluetooth.

Send "1" if person is wearing a face mask and "0" if not.
